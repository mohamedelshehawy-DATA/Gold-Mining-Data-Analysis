# Gold Mining Data Analysis

## Project Overview

This project analyzes gold exploration and mining datasets to understand mineralization patterns, evaluate gold distribution, investigate geological controls, and generate insights that support exploration targeting and mining decision-making.

The project combines geology domain knowledge with data analysis techniques using Python, Excel, and visualization tools.

## Business Problem

Mining and exploration teams need to understand where gold mineralization occurs, which geological structures are associated with higher grades, and how drillhole data can be used to support better exploration decisions.

This project aims to:

- Analyze gold grade distribution
- Investigate the relationship between gold and geology
- Identify high-grade intervals and mineralized zones
- Compare lithology / rock units and their association with mineralization
- Support exploration targeting through data-driven analysis

---

## Dataset Components

The project uses four core drillhole datasets:

- **Collar** → Drillhole collar coordinates and metadata
- **Survey** → Downhole survey data (azimuth, dip, depth)
- **Geology** → Geological intervals and rock descriptions
- **Sample** → Gold assay intervals and grades
